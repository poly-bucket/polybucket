## Additional Information

This model can be printed in multiple colors or as a single piece. Recommended print settings:
- Layer height: 0.2mm
- Infill: 15%
- Supports: Yes (for wings only)

The articulated joints use a snap-fit design that requires no additional hardware.

## Print Instructions

1. Orient the model with the belly facing down
2. Enable supports only for the wing undersides
3. Print at moderate speed (50mm/s) for best quality
4. Post-processing: Light sanding of joint surfaces recommended
